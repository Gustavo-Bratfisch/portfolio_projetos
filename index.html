Projetos em Ciência de Dados

Previsão de Churn em um Banco

Eu usei python, estatística e técnicas supervisionado de Machine Learning para criação de um modelo que consiga identificar quais clientes vão entrar churn (churn é quando o cliente não vai continuar na sua empresa). O objetivo desse modelo é reter esses clientes para continuar utilizando o serviço do banco. O resultado do modelo que construí junto com um cupom de desconto de 100 para os 100 primeiros clientes foram um retorno de R$2 milhões de reais.

Ferramentas utilizadas

  • Git e Github.
  • Pandas, Numpy, Seaborn, Sklearn.
  • Jupyter notebook.
  • Regressão Logistica, Random Forest, XGBoost.

Identificação de imóveis para compra e revenda a fim de maximizar o lucro

Nesse projeto utilizei python para identificar casas para comprar baseado nas regiões que estão inseridas.
